CONCEPTUAL DOCUMENT TEMPLATE

Conceptual documentation explain the ideas or concepts underlying a product or service. In other words, they help users understand. Unlike other types of documents, conceptual documentation does not discuss implementation details and must be concise. Examples of conceptual documentation include concept model or architecture documents.
