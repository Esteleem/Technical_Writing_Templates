REFERENCE DOCUMENTATION W/ EXAMPLE

Reference documentation is complementary to conceptual and procedural documentation. It provides additional information whenever there's friction or issues that are not discussed in other docs. Reference documentation help users get back on track. Examples of reference documentation include API Reference, Troubleshooting docs, and Release Notes.
